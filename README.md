# Bluemantle eLearning Platform

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Institutional Intelligence & Disciplinary Architecture

This repository contains the high-fidelity frontend and persistent logic for the Bluemantle eLearning platform, featuring:
- **Dual-Monitor Mission Control**: Specialized teacher oversight dashboard.
- **Persistent Orchestration**: Institutional API for real-time cross-role state management.
- **Account Disciplinary Lifecycle**: Automated suspension, formal appeal, and authority-gated reactivation workflows.
- **Advanced Scheduling**: Operational menus for batch management and multi-channel notification simulation.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
